# AI-Based Resume Ranking System

This project implements an AI-based resume ranking and shortlisting system using Natural Language Processing (NLP).

## Project Objective
To automatically rank and shortlist resumes against a given job description using TF-IDF and cosine similarity.

## Key Features
- Resume text preprocessing
- TF-IDF feature extraction
- Cosine similarity-based ranking
- Explainable skill matching
- ATS-style candidate shortlisting

## Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## Dataset
- 500 synthetic resumes
- Used only for academic and learning purposes
